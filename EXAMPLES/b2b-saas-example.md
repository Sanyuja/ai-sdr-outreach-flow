# B2B SaaS — Complete Walkthrough

Production-ready implementation for Series A-C SaaS companies.

## Results

**50 prospects over 2-3 weeks:**
- **Connection acceptance: 40%** (20 accepted)
- **Reply rate: 12%** (2-3 qualified replies)
- **Meetings booked: 2-3%** (1-2 meetings)
- **Cost: $50** in API costs
- **Time: 4 hours** total setup + 2 hours/week

## Why These Numbers

- Real research compounds acceptance (vs. templates)
- 4-message sequences catch people at different times
- LinkedIn + HeyReach is a real platform (official API partner)
- Personalization beats volume

## Implementation

1. **Source 50 companies** (Crunchbase, LinkedIn, YC directory, your network)
2. **Run enricher** (researches CEO, ICP, pain points, hooks)
3. **Review enrichment** (spot-check 5 companies for quality)
4. **Run copywriter** (generates 4 personalized messages each)
5. **Review messages** (ensure genuine personalization)
6. **Send via HeyReach** (connection requests + follow-ups)
7. **Track results** (monitor acceptance, replies, meetings)

## Example Messages (Rippling)

**Connection request:**
```
Parker, going from Zenefits to building one of the fastest-growing SaaS platforms 
in history — that's a rare second act. Would love to have you in my network.
```

**First message:**
```
HR/IT leaders are your buyers, but reaching them at scale is hard. Sales teams 
do 1:1 outreach because ads don't work for procurement. I built an AI SDR that 
finds those buyers automatically. What's your current approach?
```

**First nudge:**
```
Founders who crack GTM early automate prospecting. You've got product-market fit. 
The challenge is pipeline velocity. Curious if that's a bottleneck?
```

**Second nudge:**
```
I know you're slammed. Dropping this in case pipeline is a problem. Happy to chat 
when things slow down.
```
